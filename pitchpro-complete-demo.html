API使用例

1. 基本的な初期化

// AudioDetectionComponent初期化 const audioDetector = new AudioDetectionComponent({ // UI自動更新用セレクター frequencySelector: '#frequency-display', noteSelector: '#note-display', volumeBarSelector: '#volume-bar', volumePercentSelector: '#volume-percent', // 音響設定 fftSize: 4096, smoothing: 0.1, clarityThreshold: 0.6 }); await audioDetector.initialize();

2. コールバック設定

// カスタムコールバック設定 audioDetector.setCallbacks({ onPitchUpdate: (result) => { console.log('周波数:', result.frequency); console.log('音名:', result.note); console.log('音量:', result.volume); console.log('明瞭度:', result.clarity); }, onError: (error) => { console.error('エラー:', error); } });

3. 検出の開始と停止

// 検出開始 const success = audioDetector.startDetection(); if (success) { console.log('音程検出開始'); } // 検出停止 audioDetector.stopDetection(); // リソース解放 audioDetector.dispose();
